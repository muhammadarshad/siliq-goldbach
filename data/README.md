# Sample Data

This directory contains SMALL sample datasets for testing and validation.

## Files

- **sample_1000_pairs.csv** - First 1,000 Goldbach pairs
- **sample_10000_pairs.csv** - First 10,000 Goldbach pairs

## To Generate Full Dataset

The full 7.14 GB dataset (149,091,159 pairs) can be generated by running:

\\\ash
cd ../../src
cargo run --release > ../data/predict_100000000000.csv
\\\

**Expected:**
- Runtime: ~4 seconds on modern CPU
- Output size: 7.14 GB
- Rows: 149,091,159 pairs
- Fully reproducible (same results every run)

## Column Format

\\\csv
k,U,U_prime,d,spin,quadrant,hv
1,50000000000,50000000001,7,0,UP-,1
2,49999999999,50000000002,14,0,UP+,1
...
\\\

- **k**: Position in ring walk
- **U**: N - k (left prime)
- **U_prime**: N + k (right prime)
- **d**: Z₂₅₆ phase
- **spin**: Quadrant indicator
- **quadrant**: UP+/UP−/DN+/DN−
- **hv**: Additional metric
